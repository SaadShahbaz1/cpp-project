# 🏥 Hospital Management System (C++)

## 📌 Description
This is a very basic Hospital Management System made in C++.  
It is a simple console program that allows users to store and view patient information.

---

## 🎯 Features
- Add patient details  
- Display patient records  

---

## 🛠️ Language
- C++

---

## 🚀 How to Run
1. Compile the program:
   g++ main.cpp -o hospital  

2. Run the program:
   ./hospital  

---

## 📂 File
- main.cpp → contains the program code
